# L'Aura Restaurant — Frontend

Frontend for a Restaurant Table Reservation & Menu Management System.

## Tech Stack
- React.js, React Router
- Custom CSS (Glassmorphism UI)
- Recharts, Socket.io-client

## Features
- Multi-step reservation flow (Date → Table → Pre-order → Payment)
- Admin Dashboard with real-time revenue & booking analytics
- Live Chat widget
- Fully responsive (SPA)

## Getting Started
npm install
npm start
